# Emergency Incident Management System

## Description

This project is a **web-based Emergency Incident Management System** developed as a full-stack application.

The system allows users to report emergency incidents and enables administrators to manage and coordinate response actions. It also supports volunteer participation and real-time monitoring of active incidents.

The application is designed using a client–server architecture and includes a web interface, REST API, and database.

---

## Features

### Users

* Register and login
* Report emergency incidents
* View active incidents
* Send messages about incidents

### Volunteers

* Register and login
* Apply to participate in incidents
* View active operations
* Communicate with administrators

### Administrator

* Manage incidents
* Approve or reject incident reports
* Update incident status
* Coordinate volunteers
* View statistics

### Guests

* Report incidents without registration
* View active incidents

## Installation

### Requirements

* Node.js installed

---

### Steps

1. Extract the project files

2. Install dependencies

```
npm install
```

3. Run the server

```
node server.js
npm run dev
```

4. Open in browser

```
http://localhost:3000
```
